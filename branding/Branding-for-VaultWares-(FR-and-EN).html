BRAND GUIDE Version 2

Brand Foundation

VaultWares exists to protect individuals from pervasive tracking. We build tools where privacy is not a feature; it is the foundation.

1. Privacy

Default, not optional

No tracking. No profiling. Data minimization by design. Local-first where possible.

2. Security

In service of privacy

End-to-end encryption, self-hosted infrastructure, verifiable builds.

3. Functionality

Without compromise

Features must earn their place and never violate 1 or 2.

Core Product: VaultCore

Central API integrating identity management and AI models. Built for developers who refuse to leak user data.

Architecture

  • Self-hosted AI: All models run on VaultWares infrastructure. No third-party cloud providers (no OpenAI, Anthropic, or Google APIs in the data path).
  • End-to-end encryption: TLS 1.3 in transit. AES-256-GCM at rest. Keys never leave your tenant.
  • Optional Local Mode: Inference runs entirely on-device. Zero network calls. Perfect for sensitive workflows.
  • No tracking, no selling: We do not log prompts. We do not train on your data. Ever.

2 Correlation ID

Format: c + 6 hex chars. Use for tracing without PII.

                                
                                function newCorrelationId() {
                                  const hex = Math.floor(Math.random() * 0xffffff)
                                    .toString(16).padStart(6, '0');
                                  return `c${hex}`;
                                }
                                // example: c4f9a2b

3 Zod Validation

                                
                                import { z } from 'zod';
                                export const VaultRequest = z.object({
                                  correlationId: z.string().regex(/^c[0-9a-f]{6}$/),
                                  timestamp: z.number().int().positive(),
                                  mode: z.enum(['cloud', 'local']),
                                  payload: z.unknown()
                                });
No third-party analytics. Use server-side metrics with anonymized Correlation IDs only. No Google Analytics, Mixpanel, or similar.
